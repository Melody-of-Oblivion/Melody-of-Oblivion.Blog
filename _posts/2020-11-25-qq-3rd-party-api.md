---
title: 调用API获取QQ(群)头像
---

获取QQ号头像的接口：

```
https://qlogo4.store.qq.com/qzone/QQ号/QQ号/100
```

获取QQ群头像的接口：

```
http://p.qlogo.cn/gh/Q群号/Q群号/0
```

​    

html调用演示：

```html
<!--调用QQ号头像-->
<img src="https://qlogo4.store.qq.com/qzone/QQ号/QQ号/100">
<!--调用QQ群头像-->
<img src="http://p.qlogo.cn/gh/Q群号/Q群号/0">
```
